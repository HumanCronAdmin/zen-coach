# Zen Coach

An AI life coach grounded in Zen philosophy. No fluff, no hacks — just clarity.

## Live

https://humancronadmin.github.io/zen-coach/

## What It Does

Zen Coach is a conversational AI that helps you think through life's questions using principles from Zen philosophy. It listens, reflects, and guides — without telling you what to do.

**Target audience:** English-speaking adults interested in mindfulness, meditation, and intentional living.

## Tech Stack

- **Frontend:** Single-page HTML app with [Deep Chat](https://deepchat.dev/) UI component
- **Backend:** API hosted on Render
- **Hosting:** GitHub Pages
